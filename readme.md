# Docker with Wordpress

Install [Docker](https://docs.docker.com/install/) and run it.


## Run Wordpress

Run the command

```bash
docker-compose up
```

Open the links:
- [Wordpress](http://localhost:8080/)
- [phpMyAdmin](http://localhost:8081/)


## Directories

- `/www` All the Wordpress files
- `/mysql-cache` All the database content
